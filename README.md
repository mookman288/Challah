#Challah
##Challah.js

###Description

Challah is a prototype library for aliasing console mechanics and eventually implementing error or 
exception handling with easy to read client prompts. 

###Usage

`Challah.log(message, color, effect);`

Alias: `_cl(message, color, effect);`

`Challah.error(message, color, effect);`

Alias: `_ce(message, color, effect);`

###Example

`_cl('Hello World!');`
